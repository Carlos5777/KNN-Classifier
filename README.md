# KNN-movie-review-classifier

## Overview
This project uses a K-Nearest Neighbors (KNN) model to classify movie reviews based on their sentiment

The goal is to explore how a simple machine learning algorithm can be applied to text data and understand its performance compared to other models

## What it does
The model takes movie reviews as input, processes the text, and predicts whether the sentiment is positive or negative

## Approach
- Text preprocessing (cleaning and tokenization)
- Vectorization
- KNN classification

## Tools and libraries
- Python
- pandas
- scikit-learn

## Files in this repository
- `knn_model.ipynb` → full workflow and experiments
- `knn_model.py` → script version of the model
- `requirements.txt` → dependencies

## Results
The model is able to capture basic patterns in the text, although its performance is limited compared to more advanced models

## Notes
KNN is not the most efficient algorithm for text classification, but it is useful for understanding how distance-based models work

This project is part of my learning process in machine learning and NLP.
